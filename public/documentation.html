myFlix API

Create a REST API with Node.js

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by title /movies/[Title] GET None A JSON object holding data about a single movie, containing an ID, title, description, genre classes property.
Example:
{
   id : 1,
   title : The Irishman,
   genre : [Action, Drama]
}
Add movie /movies POST A JSON object holding data about the movie to add, structured like:
{
   title : The Irishman,
   genre : [Action, Drama]
}		 A JSON object holding data about the movie that was added, including an ID:
{
   id : 1,
   title : The Irishman,
   genre : [Action, Drama]
}
Remove a movie by ID /movies/[ID] DELETE None A text message indicating whether the movie was successfully removed
Update a movie by ID /movies/[Title]/[ID] PUT None A text message indicating the movie, title and genre was updated
Get the genres of the movie /movies/[Title]/genre GET None A text message indication the genres of the title
